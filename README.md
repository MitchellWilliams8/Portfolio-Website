# Personal Portfolio Website

A personal portfolio website built with semantic HTML, CSS, JavaScript.

## Features

* **Fully Responsive:** Adaptive layouts using CSS Grid and Flexbox.
* **High Performance:** Optimized assets, semantic HTML, and zero external framework dependencies.
* **Working Contact Form:** Integrated with Formspree (no page reloads).
* **Interactive Elements:** Smooth scrolling, hover effects, and mobile navigation.

## Tech Stack

* **HTML5** - Semantic markup
* **CSS3** - Custom Properties (Variables), Flexbox, Grid, Animations
* **JavaScript (ES6+)** - DOM manipulation, Async/Await form handling
